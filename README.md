# 🏡 WanderLust – Full-Stack Property Rental Platform

WanderLust is a full-stack property rental platform inspired by Airbnb. It allows users to explore properties, create listings, upload images, leave reviews, manage wishlists, and view property locations on interactive maps.

---

# 🌐 Live Demo

🔗 https://wanderlust-airbnb-01.onrender.com

---

# ✨ Features

- 🏠 Browse Property Listings
- ➕ Create, Update & Delete Listings
- 📸 Cloudinary Image Upload
- ❤️ Wishlist Management
- ⭐ Reviews & Ratings
- 🔍 Search Properties
- 🗂 Category Filtering
- 🗺 Interactive Maps using Mapbox
- 🔐 Secure User Authentication
- 👤 Listing Ownership Authorization
- 🛡 Joi Data Validation
- 📱 Responsive UI

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/harsht-svj/WanderLust.git
```

## Move into Project

```bash
cd WanderLust
```

## Install Packages

```bash
npm install
```

## Create .env

```env
ATLASDB_URL=your_mongodb_connection
SECRET=your_session_secret
MAP_TOKEN=your_mapbox_token

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_key
CLOUD_API_SECRET=your_cloudinary_secret
```

## Run Project

```bash
node index.js
```
